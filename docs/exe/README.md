## [cports](cports.exe)

- Windows 端口管理工具
- 使用管理员身份运行（否则可能无法强制停止端口）

## [tail](tail.exe)

- 在 Windows 中实时查看日志

```shell
tail.exe -f D:\logs\passport\passport.log
```

## [tcping](tcping.exe)

- 查看端口是否开放，同：`telnet`

```shell
tcping.exe -t www.baidu.com 80
```
